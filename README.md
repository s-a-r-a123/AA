
# AA
1. Create virtual env: `python -m venv venv`
2. Activate: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (mac/linux)
3. Install: `pip install -r requirements.txt`
4. Put `cleaned_air_data.csv` in `data/` directory.
5. Run: `streamlit run app.py`

## Deploy
Push this repo to GitHub then connect repository to Streamlit Community Cloud (https://streamlit.io/cloud).
